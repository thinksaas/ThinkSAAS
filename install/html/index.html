欢迎使用 ThinkSAAS！在您开始使用前，ThinkSAAS 需要一些数据库的信息。下列信息将会被问到，请作好准备。

  1. 数据库名
  2. 数据库用户用户名
  3. 数据库用户密码
  4. 数据库主机
  5. 表名前缀（若您希望在一个数据表中安装多个 ThinkSAAS）

大多数的互联网主机服务提供商都向您提供了数据库的信息。若您不知道这些信息，您需要先询问好，再进行安装。若您已准备好 …

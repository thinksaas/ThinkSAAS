++++ThinkSAAS installation instructions

【operating environment】
PHP>=7.0
MySQL>=5.7
Nginx>=1.15

【installation instructions】
1, upload all the files in the directory to the root directory of your website
(Linux system must use FTP binary upload)
2、Directly enter your URL to install
(Do not bring install)
3、According to the installation prompts to enter the database connection information, site information and administrator information
(Before installation, please manually create an empty database for ThinkSAAS)
4、Installation success.

【Note】

Please ensure that the following directory and all files in the directory have writable permissions (777)
cache
data
tslogs
upgrade
uploadfile
